## Getting Started

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/olesiamironenko/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser:
   - You can double-click it, or
   - Run a local server:
     ```bash
     python3 -m http.server
     ```
